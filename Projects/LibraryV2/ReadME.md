# 📚 پروژه مدیریت پیشرفته کتابخانه (C++)

---

## ✨ توضیح پروژه

این پروژه یک برنامه ساده و کاربردی به زبان **C++** است که امکان اضافه کردن **تعداد نامحدود کتاب** را فراهم می‌کند و در نهایت کتابی که **بیشترین تعداد صفحات** را دارد پیدا و نمایش می‌دهد.

---

## 🚀 ویژگی‌ها

- 📖 افزودن کتاب به صورت داینامیک (نامحدود)
- 🔁 امکان ادامه یا توقف اضافه کردن کتاب‌ها
- 📚 نمایش لیست تمام کتاب‌ها
- 🔍 پیدا کردن کتاب با بیشترین تعداد صفحات
- 🧠 استفاده از `struct` و `vector` در C++
- 💻 برنامه کنسولی ساده و کاربردی

---

## 🛠️ تکنولوژی‌ها

- زبان برنامه‌نویسی: **C++**
- کتابخانه‌ها:
  - `iostream`
  - `vector`

---

## 📌 نحوه عملکرد برنامه

1. کاربر اطلاعات یک کتاب را وارد می‌کند:
   - 📘 نام کتاب
   - ✍️ نویسنده
   - 📄 تعداد صفحات
2. کتاب در `vector` ذخیره می‌شود
3. برنامه می‌پرسد آیا کتاب دیگری اضافه شود یا نه (y/n)
4. در صورت ادامه، مراحل تکرار می‌شود
5. پس از پایان:
   - 📚 همه کتاب‌ها نمایش داده می‌شوند
   - 🔍 کتاب با بیشترین صفحات پیدا می‌شود

---

## 🧩 منطق اصلی برنامه

✔ استفاده از `vector<Book>` برای ذخیره نامحدود داده‌ها  
✔ استفاده از حلقه برای دریافت ورودی  
✔ مقایسه تعداد صفحات برای پیدا کردن بیشترین مقدار
