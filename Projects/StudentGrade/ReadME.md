# 🎓 Student Grade Manager

## 📖 معرفی پروژه

این برنامه اطلاعات دانشجویان را دریافت می‌کند و در صورتی که نمره دانشجو کمتر از 10 باشد، 2 نمره به آن اضافه می‌کند.

در پایان نمره نهایی همه دانشجویان نمایش داده می‌شود.

---

## ✨ امکانات

✅ ثبت اطلاعات دانشجویان

✅ استفاده از Struct

✅ اعمال ارفاق نمره

✅ نمایش نمرات نهایی

---

## 🛠 مفاهیم استفاده شده

- C++
- Structures
- Functions
- Pass By Reference
- Arrays

---

## 📋 نمونه اجرا

```text
Name: Ali
Grade: 8

Final Grade: 10
```

## 🎯 هدف پروژه

تمرین ساختارها (Struct)، توابع و ارسال پارامترها با ارجاع.
