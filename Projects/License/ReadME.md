# 🚗 Driving License Eligibility Checker

## 📖 معرفی پروژه

این برنامه اطلاعات متقاضیان دریافت گواهینامه رانندگی را دریافت کرده و افرادی را که شرایط سنی لازم برای دریافت گواهینامه را دارند شناسایی و نمایش می‌دهد.

حداقل سن مجاز برای دریافت گواهینامه در این پروژه ۱۸ سال در نظر گرفته شده است.
