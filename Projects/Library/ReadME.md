# 📚 پروژه مدیریت کتابخانه (C++)

---

## ✨ توضیح پروژه

این پروژه یک برنامه ساده به زبان **C++** است که اطلاعات چند کتاب را دریافت می‌کند و در نهایت **کتابی که بیشترین تعداد صفحات را دارد** پیدا و نمایش می‌دهد.

---

## 🚀 ویژگی‌ها

- 📖 دریافت اطلاعات کتاب (عنوان، نویسنده، تعداد صفحات)
- 🔍 پیدا کردن کتاب با بیشترین صفحات
- 🧠 استفاده از `struct` در C++
- 💻 برنامه کنسولی (Console Application)
- ⚡ ساده، سبک و مناسب تمرین

---

## 🛠️ تکنولوژی‌ها

- زبان برنامه‌نویسی: **C++**
- کتابخانه‌ها:
  - `iostream`
  - `string`

---

## 📌 نحوه عملکرد برنامه

1. برنامه از کاربر اطلاعات **3 کتاب** را دریافت می‌کند
2. برای هر کتاب:
   - 📘 عنوان
   - ✍️ نویسنده
   - 📄 تعداد صفحات
3. سپس بین کتاب‌ها مقایسه انجام می‌دهد
4. کتابی با **بیشترین تعداد صفحات** را پیدا می‌کند
5. نتیجه را نمایش می‌دهد
