# 🛒 Product Price Analyzer

## 📖 معرفی پروژه

این برنامه اطلاعات تعدادی محصول را از کاربر دریافت کرده و میانگین قیمت محصولات را محاسبه می‌کند.

کاربر نام و قیمت هر محصول را وارد می‌کند و در پایان میانگین قیمت تمامی محصولات نمایش داده می‌شود.
