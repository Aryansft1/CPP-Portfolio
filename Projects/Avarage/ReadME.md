# 📊 Average Analyzer

## 📖 معرفی پروژه

این برنامه مجموعه‌ای از اعداد را از کاربر دریافت کرده و میانگین آن‌ها را محاسبه می‌کند. سپس تمامی اعدادی که از میانگین بزرگ‌تر هستند را نمایش می‌دهد.

---

## ✨ امکانات

✅ دریافت تعداد اعداد

✅ محاسبه میانگین

✅ نمایش اعداد بزرگ‌تر از میانگین

✅ اعتبارسنجی تعداد ورودی‌ها

---

## 🛠 مفاهیم استفاده شده

- C++
- Dynamic Memory Allocation
- Loops
- Arrays
- Conditional Statements
