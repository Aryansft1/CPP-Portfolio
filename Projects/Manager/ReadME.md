# 👨‍💼 پروژه مدیریت کارمندان (C++)

---

## ✨ توضیح پروژه

این پروژه یک برنامه ساده و کاربردی به زبان **C++** است که اطلاعات چند کارمند را دریافت کرده و در نهایت **کارمندی که بیشترین حقوق را دارد** پیدا و نمایش می‌دهد.

---

## 🚀 ویژگی‌ها

- 👤 ذخیره اطلاعات کارمند (نام، ID، حقوق)
- 💰 پیدا کردن کارمند با بیشترین حقوق
- 🧠 استفاده از کلاس (Class) در C++
- 🏗️ استفاده از Object Oriented Programming (OOP)
- 💻 برنامه کنسولی ساده و کاربردی

---

## 🛠️ تکنولوژی‌ها

- زبان برنامه‌نویسی: **C++**
- کتابخانه:
  - `iostream`

---

## 📌 نحوه عملکرد برنامه

1. برنامه اطلاعات **3 کارمند** را از کاربر دریافت می‌کند
2. برای هر کارمند:
   - 👤 نام
   - 🆔 شناسه (ID)
   - 💰 حقوق
3. سپس بین کارمندان مقایسه انجام می‌شود
4. کارمندی که **بیشترین حقوق** را دارد انتخاب می‌شود
5. اطلاعات آن کارمند نمایش داده می‌شود
