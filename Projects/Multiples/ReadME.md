# 🔢 Reverse Multiples of Four

## 📖 معرفی پروژه

این برنامه مضرب‌های عدد 4 را در بازه 1 تا 80 به صورت معکوس نمایش می‌دهد.

در پروژه دو روش مختلف برای حل مسئله پیاده‌سازی شده است:

1. استفاده مستقیم از حلقه
2. استفاده از آرایه

---

## ✨ امکانات

✅ استفاده از دو الگوریتم متفاوت

✅ نمایش مضرب‌های 4 به صورت نزولی

✅ تمرین آرایه و حلقه‌ها
